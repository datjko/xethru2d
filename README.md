# xethru2d

## xethru_map2d

shows 2d maps of radar signals for arbitrary number of radar amplitude csv files.
radars and files must be described in a xml file of parameters (see example.xml).

## ampl2histogram

generates histograms of radar signals from given set of radar amplitude csv files.

## dependencies

requires boost (tested with version 1.54) and opencv (tested with version 2.4.8, should work with any)
